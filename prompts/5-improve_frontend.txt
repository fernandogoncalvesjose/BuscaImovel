- Filtros
- Página de detalhes
- Integração frontend/backend

Agora quero melhorar a experiência do MVP para que ele pareça mais profissional e gere mais valor para quem busca imóvel.

Objetivo da Etapa 5:
Melhorar a UX da busca e criar pequenos diferenciais que ajudem o usuário a decidir melhor.

1. Ordenação dos imóveis
Adicionar opções:
- Menor preço

- Menor custo mensal
- Maior área
- Menor preço por m²
- Mais recentes

Nos cards e detalhes, destacar:
- preço
- condomínio
- IPTU
- custo mensal total

O custo mensal total deve aparecer como uma informação de decisão, não apenas como texto pequeno.

3. Badges inteligentes nos cards
Criar badges como:
- Aceita pet
- Mobiliado
- Bom custo-benefício
- Menor preço/m²
- Novo anúncio
- Baixo condomínio

As badges podem ser calculadas no frontend inicialmente.

4. Melhorar filtros
Adicionar:
- botão "Limpar filtros"
- contador de imóveis encontrados
- filtros colapsáveis no mobile
- estado visual para filtros ativos

5. Melhorar Empty State
Quando não encontrar imóveis, mostrar:
- mensagem amigável
- sugestão para remover filtros
- botão "Limpar filtros"

6. Melhorar página de detalhes
Adicionar uma seção chamada:
"Resumo inteligente"

Esse resumo deve mostrar:
- custo mensal total
- preço por m²
- perfil do imóvel
- principais pontos positivos

- Preço
